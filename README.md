Tabletop Simulator Modding Hub

A comprehensive collection of Lua scripts, assets, and frameworks designed to enhance, streamline, and expand the modding capabilities within Tabletop Simulator. This repository serves as a central hub for various projects, from world-building systems to game-specific mods.
Featured Projects
One World System

A revolutionary framework for creating seamless, interconnected game worlds. Dynamically load and unload zones without interrupting gameplay, perfect for large-scale RPGs, exploration games, and complex scenarios.

    Core Features: Zone management, asset streaming, state persistence
    Components: VBase.lua, WBase.lua, MBag.lua, OneWorld.lua
    Ideal for: Large RPG campaigns, exploration modules

Fallout System

A specialized scripting suite for creating tabletop experiences in the Fallout universe. Includes character management, item systems, and thematic mechanics.

    Core Features: Character sheets, S.P.E.C.I.A.L. system, inventory management
    Components: Character sheets, item cards, rule systems
    Ideal for: Fallout-themed tabletop campaigns

Utility Libraries

Reusable code snippets and helper functions to accelerate your own mod development.

    Core Features: UI helpers, data management, object manipulation
    Components: Common functions, templates, examples
    Ideal for: All Tabletop Simulator modders

Repository Structure
/Tabletop_Simulator_Modding /OneWorld - Core system for seamless world management /Fallout_System - Complete Fallout tabletop implementation /Utilities - Reusable helper functions and templates /Assets - Images, models, and other resources /Docs - Documentation and tutorials
Quick Start

    Clone or download the repository
    Import the relevant Lua scripts into your Tabletop Simulator objects
    Follow the inline documentation and comments for configuration
    Check the /Docs folder for detailed tutorials

Requirements

    Tabletop Simulator (Steam)
    Basic understanding of Lua scripting
    For advanced use: familiarity with TTS object model and API
